# 03_cicd_ecs_pipeline

CI/CD パイプライン（CodePipeline + ECS）。

## 構成サービス

- CodePipeline
- CodeBuild
- CodeDeploy (Blue/Green)
- ECR
- ECS Fargate
- CDK (TypeScript)

## 学べること

- CodePipeline の各ステージ設計
- CodeBuild でのイメージビルド・プッシュ
- ECS Blue/Green デプロイの仕組み
- IAM 設計（最小権限）
- 失敗時のロールバック戦略
